# Contributing To ZeroWall 🛡️

Thank you for your interest in contributing to ZeroWall! As an open-source project, we welcome contributions from the community to make Linux firewall management simpler and more accessible.

## 🚀 How to Contribute

### 1. Reporting Bugs
- Use the [GitHub Issues](https://github.com/boniyeamincse/zerowallcli/issues) tracker.
- Provide a clear description of the issue and steps to reproduce it.
- Include your environment details (OS, Python version).

### 2. Suggesting Features
- Open an issue with the tag `enhancement`.
- Describe the feature and why it would be useful.

### 3. Code Contributions
- Fork the repository.
- Create a new branch for your feature or fix (`git checkout -b feature/awesome-feature`).
- Ensure your code follows the existing style and is well-documented.
- Run tests (`sudo python3 tests/test_firewall.py`) before submitting.
- Submit a Pull Request (PR) with a clear description of your changes.

## 🛠️ Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/zerowallcli.git
cd zerowallcli

# Install dependencies
sudo ./scripts/install.sh
```

## 📜 Code of Conduct
Please be respectful and professional in all interactions within the ZeroWall community. We aim to foster a welcoming environment for everyone.
